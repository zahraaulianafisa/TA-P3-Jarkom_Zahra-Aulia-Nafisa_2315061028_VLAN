Tugas ini merupakan tugas akhir praktikum mata kuliah Jaringan Komputer dengan tujuan untuk menerapkan konsep Inter-VLAN Routing menggunakan metode Router on a Stick pada Cisco Packet Tracer. Konfigurasi dilakukan agar perangkat-perangkat dari VLAN yang berbeda dapat saling berkomunikasi melalui router. Dalam konfigurasi ini juga diterapkan teknik trunking antar switch untuk memungkinkan lalu lintas beberapa VLAN melewati satu jalur fisik.

Tahapan konfigurasi mencakup :
- Pembuatan dan penamaan VLAN pada masing-masing switch.
- Pengaturan port access untuk setiap VLAN.
- Pembuatan trunk antar switch dan antara switch dengan router.
- Penerapan sub-interface pada router sebagai gateway untuk setiap VLAN.
- Pengujian konektivitas antar perangkat menggunakan perintah ping.

Percobaan ini bertujuan untuk memahami cara kerja komunikasi antar VLAN serta memastikan bahwa routing antar jaringan logis dapat berjalan dengan baik.
